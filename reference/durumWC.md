# durumWC

200 sites from durum wheat collection and their world clim data.

## Usage

``` r
data("durumWC")
```

## Format

The data includes the site unique identifier, longitude, latitude and 55
[worldclim data](https://www.worldclim.org/data/worldclim21.html)

## Examples

``` r
if(interactive()){
 # Load durum wheat data with world climatic variables obtained from WorldClim database
 data("durumWC")
 }
```
