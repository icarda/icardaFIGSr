# durumDaily

200 sites from durum wheat collection and their daily climatic data.

## Usage

``` r
data("durumDaily")
```

## Format

The data includes the site unique identifier and daily data for 4
climatic variables (tmin, tmax, precipitation and relative humidity)

## Examples

``` r
if(interactive()){
 # Load durum wheat data with their daily climatic variables
 data("durumDaily")
}
```
