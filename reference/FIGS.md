# FIGS

FIGS subset for wheat sodicity resistance constructed using the FAO
harmonized world soil database HWSD

## Usage

``` r
data("FIGS")
```

## Format

A data frame with 201 rows and 15 variables

## References

[HWSD](https://openknowledge.fao.org/server/api/core/bitstreams/2f45beea-e165-4264-951e-20b7c1da65c0/content)

## Examples

``` r
if(interactive()){
 data("FIGS")
}
```
