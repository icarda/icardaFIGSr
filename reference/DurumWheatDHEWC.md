# DurumWheatDHEWC

193 samples of Days to heading from durum wheat collection and 55
worldclim data.

## Usage

``` r
data("DurumWheatDHEWC")
```

## Format

More details about the climate data can be found here
[worldclim](https://www.worldclim.org/data/worldclim21.html)

## Examples

``` r
if(interactive()){
 # Load durum wheat Days to Heading data with climatic variables
 # obtained from WorldClim database
 data("DurumWheatDHEWC")
 }
```
